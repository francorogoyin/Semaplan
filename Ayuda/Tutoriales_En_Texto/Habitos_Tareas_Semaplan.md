# Usar Habitos y Tareas en Semaplan.

Video introductorio para mostrar los paneles de habitos y tareas sin modificar datos.

## Pasos.

1. Abrir Habitos.
   Entrar desde el menu superior y revisar el panel.
2. Abrir Tareas.
   Cambiar al panel de tareas para revisar pendientes.

## Guion de narracion.

Habitos sirve para registrar rutinas repetibles y medir consistencia.

Tareas muestra pendientes operativos y acciones concretas separadas del plan general.
