# Usar Metas en Semaplan.

Recorrido lento por metas de período, progreso, subobjetivos y avances.

## Pasos.

1. Abrir Metas desde Planes.
   Entrar a la vista de período y ubicar los controles principales.
2. Revisar progreso.
   Leer tarjetas, porcentajes, unidades y estados.
3. Registrar avance.
   Abrir el formulario de avance y ver qué datos pide.

## Guion de narracion.

En este video vamos a recorrer Metas dentro de Planes. Sirven para seguir objetivos de más de una semana, medir progreso y conectar avances con períodos concretos.

Abrimos Planes para entrar a la zona donde viven las metas de período.

La capa define el tamaño del período: año, trimestre, mes u otra escala disponible.

El subperíodo permite moverse dentro del año sin perder el contexto.

El filtro de estado separa metas activas, cumplidas, pausadas u ocultas.

El cambio de vista permite mirar las metas como tarjetas, lista o biblioteca.

El resumen del período explica para qué sirve este tramo y qué se está intentando lograr.

Cada tarjeta de meta muestra nombre, unidad, progreso y señales de ritmo.

Abrimos Registrar avance para cargar progreso real sobre una meta.

Primero se elige a qué objetivo, subobjetivo o parte pertenece el avance. El selector muestra la opción actual y permite cambiarla.

Después se carga una cantidad. La unidad depende de la meta: horas, sesiones u otra unidad definida.

La fecha ubica el avance dentro del período correcto.

Cancelamos el formulario para no duplicar un avance durante el tutorial.

Cuando se guarda un avance real, la tarjeta recalcula el progreso de la meta.

Cerramos Planes y dejamos las metas preparadas para seguir usándolas.
