# Usar el Baul de objetivos de Semaplan.

Video introductorio para mostrar objetivos no programados, filtros, categorias y vistas del Baul.

## Pasos.

1. Abrir el Baul.
   Usar el boton del menu superior.
2. Revisar filtros.
   Mirar busqueda, categorias, etiquetas, estado y periodo.
3. Cambiar vista.
   Usar los controles de biblioteca o lista.

## Guion de narracion.

El Baul guarda objetivos que todavia no estan puestos en la semana.

Se puede filtrar por categoria, etiqueta, estado, periodo y cambiar entre vista biblioteca y lista.
