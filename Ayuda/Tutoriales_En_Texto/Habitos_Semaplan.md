# Usar Hábitos en Semaplan.

Recorrido lento por hábitos, registros rápidos, vistas y creación.

## Pasos.

1. Abrir Hábitos.
   Entrar al panel y revisar hábitos activos.
2. Registrar avances.
   Usar los botones rápidos y cambiar entre día, semana y mes.
3. Crear un hábito.
   Guardar una rutina nueva con meta medible.

## Guion de narracion.

En este video vamos a recorrer Hábitos. Esta sección sirve para sostener rutinas, registrar avances diarios y ver si una conducta se está cumpliendo con constancia.

Abrimos Hábitos desde el menú principal.

Cada hábito tiene un nombre, una meta y un estado. Este ejemplo pide tomar agua dos veces por día.

El botón rápido registra un avance sin abrir formularios. Es útil para hábitos simples y repetidos.

Cambiamos a vista semanal para ver continuidad, no solo el día actual.

La vista semanal ayuda a detectar patrones: qué se repite, qué falta y qué conviene ajustar.

La vista mensual sirve para revisar una tendencia más larga.

Cuando el período cambia, Semaplan recalcula el progreso con el mismo hábito y sus registros.

El Registro muestra el historial de avances cargados.

Esta vista sirve para revisar cuándo se registró cada hábito y corregir el seguimiento si hace falta.

Volvemos al panel principal antes de crear un hábito nuevo.

Abrimos el editor para crear una nueva rutina.

Escribimos un hábito con una acción fácil de entender.

Elegimos un emoji para reconocerlo rápido en el panel.

La meta puede ser por check, cantidad o tiempo. Para respirar, usamos tiempo.

La cantidad define cuánto cuenta como avance suficiente.

Guardamos el hábito. Desde ahora aparece en el panel y puede registrarse como cualquier otro.

Hábitos combina rutina, medición y revisión histórica en una sola sección.

Cerramos Hábitos con los cambios guardados.
