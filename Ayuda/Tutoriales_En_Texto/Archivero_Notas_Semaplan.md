# Usar el Archivero de Semaplan.

Video introductorio para mostrar cajones, notas, busqueda, filtros y creacion de notas en el Archivero.

## Pasos.

1. Abrir el Archivero.
   Usar el boton del menu superior.
2. Elegir un cajon.
   Mirar la lista lateral y el contenido del cajon abierto.
3. Crear una nota.
   Abrir el modal de nueva nota sin guardar cambios.

## Guion de narracion.

El Archivero sirve para guardar notas, referencias y material operativo dentro de cajones.

Desde la cabecera se puede buscar, filtrar, exportar el cajon y abrir una nueva nota.
